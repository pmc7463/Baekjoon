* 2022-08-18 목요일
    + 2753 윤년 문제

    ``` 
    if(year % 4 == 0){              // 1단계
        if(year % 100 == 0) {       // 2단계
            if(year % 400 == 0){    // 3단계
                printf("1");        // 최종 윤년
            }
            else
                printf("0");        // 3단계 윤년 아님
        }
        else
            printf("1");            // 2단계 윤년
    }
    else 
        printf("0");                // 1단계 윤년 아님
    ```
    윤년을 정하는 단계가 있다는것을 알게 되었다.
